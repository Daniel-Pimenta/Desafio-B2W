# b2w-rest-api

A long time ago in a galaxy far far away

Desafio B2W

Esse pequeno sistema disponibiliza serviços rest

./api/planeta/list -> Lista todos os planetas no banco de dados
./api/planeta/get/id/{id} -> Lista todos o planeta no banco de dados para o ID informado
./api/planeta/get/nome/{nome} -> Lista todos o planeta no banco de dados para o nome informado
./api/planeta/edit/{id} -> Alatera o planeta no banco de dados para o ID informado
./api/rest/planeta/add -> Adiciona um planeta no banco de dados
./api/planeta/delete/{id} -> Deleta o planeta no banco de dados para o ID informado

O banco de dados é o MongoDB (cloud)

mongodb+srv://b2w:b2w@cluster0-is1wv.mongodb.net

database => b2w
collection => planetas
O documento segue o formato

{
    "id": 1,
    "nome": "Hoth",
    "clima": "MUUUUITO FRIO",
    "terreno": "DURO",
    "qtdAparicao": "1"
}

Para a Inserção de Dados foi utilizado o Aplicativo Postman
